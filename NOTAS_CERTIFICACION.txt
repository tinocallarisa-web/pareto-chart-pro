Free tier (no license required):
- Fixed 10% bins (10 bars showing top → bottom entities)
- Full Pareto chart with cumulative % line
- Up to 2 configurable reference lines (default: 80%)
- Bar color, opacity, cumulative line color settings
- Cross-filtering (filter-out via click, filter-in via highlights)
- Tooltips and context menu

Pro tier (requires active AppSource license):
- Custom bin size: 1%–20% (default 5%)
- Outlier filter: exclude top/bottom % of entities
- Bar border color and width
- Bar gap control
- Value labels on bars (font size, color, show % sign)
- Third reference line
- All Free features included

Testing instructions:
1. Import the .pbiviz file from the submission package
2. Add an Entity field (e.g. customer_id) and a Value field (e.g. sales)
3. WITHOUT Pro license: verify 10 bars appear (10% bins);
   badge shows "Free: 10% bins — upgrade to Pro for custom bin size..."
   Pro-only settings (bin size, trim, value labels) do NOT appear in Format Pane
4. WITH Pro license key below: verify Format Pane shows Pro settings
   (Bins section, Outlier Filter, Value Labels)
   Change bin size to 5% → 20 bars appear, correctly decreasing
5. Cross-filtering: click a bar → other visuals on page filter
6. Filter-in: click another slicer → unselected bins dim to 25% opacity
7. Right-click a bar → Power BI context menu appears (Include/Exclude/Drill)
8. Hover a bar → tooltip shows: Entity range, Count, % of total, Cumulative %
9. Reference lines: default 80% horizontal + vertical crosshair mark crossing point
   Enable line 2, set to 60% → second crosshair appears in different color
