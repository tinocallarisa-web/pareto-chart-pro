Getting Started

Drag a Category field and a Value (numeric) field into the visual. Bars will be sorted automatically from highest to lowest value with a cumulative percentage line overlaid.

FAQ

Why do I only see 10 categories?
The Free tier shows up to 10 categories. Upgrade to Pro to display more.
How do I change colors?
Open the Format Pane → Chart Settings. You can set the bar color, line color, and 80% reference line color independently.
Can I hide the 80% reference line?
Yes — toggle "Show 80% Reference Line" off in the Format Pane.
Does cross-filtering work?
Yes. Click a bar to filter other visuals. Click on other visuals to dim non-selected bars in this chart.
The visual shows a blank canvas
Make sure you have assigned both a Category and a Value field. The Value field must be numeric.
